# liushangzuo.com

柳尚佐的个人网站 — 记录生活，分享思考。

**线上地址：** https://liushangzuo.com

## Tech Stack

- **Framework:** Nuxt 3 (SSR)
- **Styling:** Tailwind CSS v4
- **Content:** @nuxt/content (Markdown)
- **CMS:** Nuxt Studio
- **Deployment:** Vercel

## Development

```bash
pnpm install
pnpm dev
```
